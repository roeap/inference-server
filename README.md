# inference-server

Playground for an experimental inference server based on [V2 inference protocol][v2] and the [onnx][onnx].

## Development

### Prerequisites

- Poetry `>= 1.3`
- just
- rust toolchain

```sh
just install
```

[v2]: https://kserve.github.io/website/modelserving/inference_api/#grpc
[onnx]: https://onnx.ai/
